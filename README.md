# N-Queens Solver

A graphical application built in Python to solve the classic N-Queens puzzle. 
This program visualizes the placement of N queens on an N×N chessboard such that no two queens threaten each other.

## Features

* **Visual Board Rendering:** Renders a dynamic chessboard based on the user-provided value of N.
* **Interactive Solving:** Allows users to input the number of queens and find all possible solutions.
* **Solution Navigation:** Once calculated, users can browse through all valid solutions using "Previous" and "Next" buttons.
